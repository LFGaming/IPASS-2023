\doxysubsection{Detailed Description}
Class for interfacing with the \mbox{\hyperlink{class_b_m_p280}{BMP280}} temperature and pressure sensor. 

The \mbox{\hyperlink{class_b_m_p280}{BMP280}} class provides functions to communicate with the \mbox{\hyperlink{class_b_m_p280}{BMP280}} sensor using I2C. It allows reading temperature and calibration data from the sensor.
