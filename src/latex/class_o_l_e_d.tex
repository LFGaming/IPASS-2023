\doxysubsection{Detailed Description}
Class for interfacing with the \mbox{\hyperlink{class_o_l_e_d}{OLED}} display. 

The \mbox{\hyperlink{class_o_l_e_d}{OLED}} class provides functions to communicate with the \mbox{\hyperlink{class_o_l_e_d}{OLED}} display using I2C. It allows adding a header line, printing temperature and humidity values, and clearing the screen.
