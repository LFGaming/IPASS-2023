\doxysubsection{Detailed Description}
Class for interfacing with the \mbox{\hyperlink{class_s_i7021}{SI7021}} temperature and humidity sensor. 

The \mbox{\hyperlink{class_s_i7021}{SI7021}} class provides functions to communicate with the \mbox{\hyperlink{class_s_i7021}{SI7021}} sensor using I2C. It allows reading temperature and humidity data from the sensor.
